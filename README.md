# justfood

A web app that provides cooking recipes. Made using Kuroco-Front and Nuxtjs-3.
Technologies Used: Nuxtjs, Vue, Js, SCSS

## Deployment
https://sample-jassi.g.kuroco-front.app

## Running the app
```
npm install
npm run dev
```
